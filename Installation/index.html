1. Clone files

git clone https://github.com/KrowNetwork/KrowAlpha

2. Run install script

From the folder that git clone made, run:
chmod +x install.sh
./install.sh

3. Start the model

For the first time only, run the following command:*
chmod +x startFabric.sh
./startFabric.sh

You shouldn't have to run that command ever again unless the server turns off
