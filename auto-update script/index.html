If you have never ran the auto update script, navigate to the KrowAlpha folder, and run the following:
chmod +x auto-update.sh

To run:
./auto-update.sh BRANCH PORT

Parameters:
1. BRANCH - github branch to use (master, etc)
2. PORT - port to run on (8888, 5555, 3333, etc)

Example:
./auto-update.sh master 8888
